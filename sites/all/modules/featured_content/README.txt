ADDING FEATURED CONTENT BLOCKS

    To add new Featured Content Blocks, use the "Add Featured Content Block" tab 
    (or button) on the administer blocks page, admin/build/block. You will 
    then be able to configure your featured content block before adding it.

CONFIGURING FEATURED CONTENT BLOCKS

    When adding or configuring a Featured Content Block, several configuration 
    options are available. The FC block form is fully documented but here is a
    quick summary of what is on the form.

    * Block title - title to show for the block

    * Block name - name used internally on blocks page

    * Header Text - text to show between title and block content

    * Footer Text - text to show after block content

    * Empty Text - text to show if there is no block content

    * Number to Show - number of content items to show

    * Sort By - how to sort the content

    * What to Display in Block - style for the content in block, e.g. titles vs teasers

    * Display Style - if titles, display as ordered or unordered list

    * What to Display on Read More Page - read more page options

    * Include RSS Feed - whether to include an rss feed icon and page

    * Type: manual, CCK, filtered, or search

    > For manual:

    * URLs - manually selected internal URLs

    > For CCK:

    * CCK Fields - fields that contain node ids or references

    > For filtered:

    * Include Node Being Viewed - show current node in content block
 
    * Content Types - which content types to use when filtering content

    * Users - which users to use when filtering content

    * Terms - which vocabulary terms to use when filtering content

    * Path - which path parts to use when filtering content

    * Keywords - if title keywords should be used when filtering content

    * Views - if views results should be used

    * Primary Term - if primary term should be used when filtering content

    > For search:

    * Include Node Being Viewed - show current node in content block

    * Num Words in Title - how many of the title words should be used for search
